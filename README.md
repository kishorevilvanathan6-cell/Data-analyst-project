# IPL 2025 season batting card project
## Overview
Analyze IPL batting data to identify score to understand the performances.

## ABSTRACTION
THIS PROJECT ANALYSIS THE IPL BATTERS 2025
THE DATASET CONTAIN PLAYERS INDIVIDUAL SCORES, HALF CENTURIES, FULL CENTURIES AND BOUNDARIES OF IPL 2025
THE NULL VALUES OF THIS PROJECT IS POSSIBLE DATA
THIS ANALYSIS HELPED IDENTIFY TOP PERFORMANCES OF THE BATTERS TEAM IN IPL 2025
